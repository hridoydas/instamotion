## Run Locally

Clone the project

```bash
  git clone https://github.com/hridoydas/instamotion
```

Go to the project directory

```bash
  cd instamotion
```

Install dependencies

```bash
  npm Install
```

Run the gherkin syntex

```bash
  npm run gherkin
```

Run the test cases
```bash
  npm run test
```
